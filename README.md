# gulp pack
=================
It's a start pack of gulp + express.

# What's in this pack
  * tool
    * gulp
    * bower
  * preprocessor
  	* pug
    * stylus
    * LiveScript
  * webserver
  	* express with livereload server
  * frontend module
    * jQuery
    * semantic-ui


# setup
  * yarn (or npm i)
  * echo '[express running port number]' > port

# command

## run express
  * yarn start (or npm start)

## build static file
  * yarn run build (or npm run build)
